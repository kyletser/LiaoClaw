# LiaoClaw 项目交付运行手册（Windows + macOS）
本项目是参照pi-mono（类似openclaw的核心部分）的方式设计的编程智能体，包含Web、CLI、IM三种交互方式，具体功能参照README.md。
推荐在智谱ai开放平台注册，新人送2000w 免费token,本项目默认使用智谱ai的glm-4.5-air模型（anthropic兼容调用），可自行修改。
智谱平台创建apikey之后，在`.env.ps1`（Windows）中填入：
`$env:ANTHROPIC_API_KEY = "你的key"`

飞书配置部分可参照：https://www.runoob.com/ai-agent/openclaw-feishu.html 中的飞书配置部分，从在飞书开放平台创建机器人这一步开始。

https://www.runoob.com/ai-agent/openclaw-clawdbot-tutorial.html
这是详细的openclaw配置教程，推荐尝试。

## 1. Windows 一分钟启动（最短可用）

在当前项目根目录执行：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -e ".[dev]"
Copy-Item .env.ps1.example .env.ps1
```


Windows：

1. Web：`liaoclaw-web`
2. CLI：`.\dev.ps1 -Mode cli `
3. IM 长连接（飞书）：`.\dev.ps1 -Mode im -Transport longconn`



## 2. 什么时候用长命令

仅在默认值不满足时加参数，例如端口冲突：

```bash
liaoclaw-web --provider anthropic --model-id glm-4.5-air --host 127.0.0.1 --port 8788
```

说明：`liaoclaw-web` 默认是 `127.0.0.1:8787`。


## 3. macOS 一分钟启动（最短可用）

在当前项目根目录执行：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
cp .env.example .env
```

macOS：

1. Web：`liaoclaw-web`
2. CLI：`chmod +x ./dev.sh && ./dev.sh --mode cli`
3. IM 长连接（飞书）：`chmod +x ./dev.sh && ./dev.sh --mode im --transport longconn`
