# LiaoClaw 项目交付运行手册（Windows 简版优先）

适用场景：把项目完整发给别人，对方在 Windows 上直接运行。

## 1. 一分钟启动（最短可用）

先在项目根目录执行：

```powershell
py -3.10 -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -e ".[dev]"
Copy-Item .env.ps1.example .env.ps1
```

然后编辑 `.env.ps1`，至少填一个 Key：

1. `$env:ANTHROPIC_API_KEY = "你的key"`  
2. 或 `$env:OPENAI_API_KEY = "你的key"`

启动 Web（简短版）：

```powershell
liaoclaw-web --provider anthropic --model-id glm-4.5-air
```

浏览器打开：`http://127.0.0.1:8787`

## 2. 简短版命令（推荐）

1. Web：`liaoclaw-web --provider anthropic --model-id glm-4.5-air`
2. CLI：`. .\.env.ps1; liaoclaw --mode interactive --provider anthropic --model-id glm-4.5-air`
3. 开发脚本（CLI）：`.\dev.ps1 -Mode cli`

## 3. 什么时候用长命令

只有在默认值不满足时才加参数，比如端口冲突：

```powershell
liaoclaw-web --provider anthropic --model-id glm-4.5-air --host 127.0.0.1 --port 8788
```

说明：`liaoclaw-web` 默认就是 `127.0.0.1:8787`，所以大多数人不需要写 `--host` 和 `--port`。

## 4. 可选：飞书 IM（Windows）

先安装：

```powershell
pip install -e ".[dev,feishu]"
```

再在 `.env.ps1` 填写：

1. `$env:FEISHU_APP_ID`
2. `$env:FEISHU_APP_SECRET`
3. `$env:FEISHU_VERIFY_TOKEN`（Webhook 可选）

长连接（最短推荐）：

```powershell
.\dev.ps1 -Mode im -Transport longconn
```

Webhook（最短推荐）：

```powershell
.\dev.ps1 -Mode im -Transport webhook
```

需要精确指定参数时，再用长命令：

```powershell
liaoclaw-im --platform feishu --transport longconn --provider anthropic --model-id glm-4.5-air --feishu-app-id $env:FEISHU_APP_ID --feishu-app-secret $env:FEISHU_APP_SECRET
```

## 5. 常见问题

1. `Missing ANTHROPIC_API_KEY`：没配 Key，补 `.env.ps1`。
2. `Unknown model`：模型名写错，先用 `anthropic + glm-4.5-air`。
3. `WinError 10053`：浏览器断开连接常见日志，一般不影响使用。
4. `Address already in use`：端口占用，换 `--port 8788`。
5. `liaoclaw 不是内部或外部命令`：没激活 `.venv` 或没安装依赖。

## 6. 交付前自检清单

1. `python --version` 是 `3.10+`
2. `pip install -e ".[dev]"` 成功
3. `.env.ps1` 已填真实 Key
4. `liaoclaw-web --provider anthropic --model-id glm-4.5-air` 可启动
5. 页面能正常对话
